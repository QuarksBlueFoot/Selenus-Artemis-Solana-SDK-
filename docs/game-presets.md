# Game presets

These presets provide sane defaults for Conduit publishing across game genres.

They control:
- commit cadence
- snapshot cadence
- backpressure defaults

## Presets

- `MMORPG_RPG`
- `PLATFORMER`
- `SHOOTER`
- `IDLE`

## Example

```kotlin
val preset = GamePreset.MMORPG_RPG
```

Use:

```kotlin
val preset = GamePreset.MMORPG_RPG
```
